Start developing in minutes

Our open source SDKs in most of today's popular languages and frameworks let you get up and running in minutes. Just generate an API key and start pushing data.

Our most active SDKs are JavaScript, Ruby and Go - but we have many more options available on GitHub. We encourage issues, suggestions and contributions!

  • JavaScript
  • Ruby
  • Go
  • REST
stretchr = new Stretchr.Client("project.company", "key");

	stretchr.at("people/1").read({
		success: function(response){
		alert("Hello " + response.resource().data("name"));
	}
});
stretchr = Stretchr::Client.new(
	project: "project.company",
	key: "key")

Stretchr := stretchr.NewSession("{project}", "{key}");

// read a resource
resource, _ := Stretchr.At("people/123").ReadOne()

log.Printf("Just loaded %s who is %g years old.", resource.Get("name"), resource.Get("age"))

Stretchr's RESTful API allows easy development in any language, even those we don't yet support through native SDKs.

Your data, completely future-proofed

Anyone who's worked with NoSQL before knows how easy it is to shoot yourself in the foot with schemaless data. You'll often encounter enterprise deployments riddled with data duplication, lost relations and lost querying capabilities.

Stretchr's nested data model and powerful query interface let you cut and slice your data however you need for any situation. Just send your data in as it appears in the real world and we'll keep it accessible for future development.

Built for monetizing data

Our unique permissions system lets you layer multiple levels of permissions on your data. Set which segments of data each application can access, including limitations based on which user is using the app. This fine-grained control over data exposure enables key monetization channels such as third party development, customer-facing apps, mobile apps and web applications. All without any additional backend development, these channels can securely connect directly to your datastack.

Just one example of where this becomes useful is in the development of customer dashboards and internal support applications.
